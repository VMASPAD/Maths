Para resolver una función racional, se pueden seguir los siguientes pasos. A continuación, se resolverá un ejemplo paso a paso:

$$x = 2$$

Encontrar los puntos de corte con los ejes. Para encontrar el punto de corte con el eje x, se debe resolver la ecuación f(x) = 0 En este caso, la ecuación es
